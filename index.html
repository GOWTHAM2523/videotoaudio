How to Use

  • Drag and drop a video file into the designated drop zone or click to select a file.
  • Wait for the file to be uploaded and recognized.
  • Click the "Convert to Audio" button to start the conversion process.
  • Once the process is complete, rename the file if desired.
  • Click the "Download MP3" button to save the audio file to your device.
